## About

Integrates Spring Boot REST APIs with JPA (Java Persistence API) for database operations. This assignment connects RESTful endpoints to a relational database using entity mapping and repository patterns.

## Key Concepts

- JPA entity annotations (`@Entity`, `@Table`, `@Id`, `@Column`)
- `EntityManager` for persistence operations
- Connecting REST controllers to database layer
- Application properties for database configuration
